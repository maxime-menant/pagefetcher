== Page Fetcher

This project was done for an interview Test.

Page Fetcher save a Facebook page ID into its database, and list all the pages saved. For each page, it fetch the last 10 posts on the page's feed.

== Time

This project was done in two steps, one of 3hours, and the second one of 1.5hours

== Test

This project is tested with RSpec for models and controllers.
